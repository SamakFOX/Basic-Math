# 코딩테스트를 위한 기초수학
알고리즘 테스트에서 첫문항으로 자주 출제되는 수학적 문제를 해결하기 위해, 초/중/고 기초수학을 이해하고 활용할 수 있어야 합니다.  
※ SSAFY의 SW 기초역량 테스트에서 수1(고2 과정)~대학수학 내용이 출제된 사례가 있습니다.
