# 약수와 배수

### 01. 약수와 배수의 정의
약수 : 어떤 수를 나누어 떨어지게 하는 수  
배수 : 어떤 수를 n배 한 수  
> A x B = C 일때 A,B는 C의 약수이며 C는 각각 A와 B의 배수이다.

| | 예시 |
|---|---|
| 8의 약수 | 8 ÷ 1 = 8, 8 ÷ 2 = 4, 8 ÷ 4 = 2, 8 ÷ 8 = 1 |
| | 1, 2, 4, 8 |
| 2의 배수 | 2 x 1 = 2, 2 x 2 = 4, 2 x 3 = 6, 2 x 4 = 8 |
| | 2, 4, 6, 8 ··· |

### 02. 공약수와 최대공약수
공약수 : 두 수의 공통된 약수
최대공약수 : 두 수의 공약수 중 가장 큰 약수
> 최대공약수의 약수는 두 수의 공약수와 같음

| 예시 | 12, 18 |
|---|---|
| 12의 약수 | 1, 2, 3, 4, 6, 12 |
| 18의 약수 | 1, 2, 3, 6, 9, 18 |
| 공약수 | 1, 2, 3, 6 |
| 최대공약수 | 6 |

※ 소인수분해로 최대공약수 구하는 방법
1) 두 수를 소인수분해  
> 12 = 2² × 3¹  
> 18 = 2¹ × 3²  
2) 공통으로 포함된 소인수의 최소 제곱수만 곱하기  
> 2: 12는 2², 18은 2¹ → 공통은 2¹  
> 3: 12는 3¹, 18은 3² → 공통은 3¹  
> 2¹×3¹ = 2×3 = 6
