# 분수
a를 0이 아닌 b로 나눈 몫을 $\frac{a}{b}$ 또는 a/b로 표시한 것

### 01. 분모가 같은 분수의 덧셈과 뺄셈
> 분모가 같은 분수의 연산에서는 분모는 그대로, 분자만 연산

| 예시 |
|---|
| $\frac{2}{5}$ + $\frac{3}{5}$ = $\frac{2+3}{5}$ = $\frac{5}{5}$ = 1 |
| $\frac{6}{8}$ - $\frac{2}{8}$ = $\frac{6-2}{8}$ = $\frac{4}{8}$ = $\frac{1}{2}$ |

### 02. 약분
분모와 분자를 공약수로 나누는 것  
※ 기약분수 : 분모와 분자의 공약수가 1뿐인 분수  
> 한번의 약분으로 기약분수를 만들기 위해서는 두 수의 최대공약수로 나눠줌

| 예시 |
|---|
| $\frac{54}{90}$ 약분 |
| 54의 약수 : 1, 2, 3, 6, 9, 18, 27, 54<br> 90의 약수 : 1, 2, 3, 5, 6, 9, 10, 15, 18, 30, 45, 90<br> 공약수 : 1, 2, 3, 6, 9, 18 |
| $\frac{54}{90}$ = $\frac{54÷2}{90÷2}$ = $\frac{27}{45}$ = $\frac{54÷3}{90÷3}$ = $\frac{18}{30}$ = $\frac{54÷6}{90÷6}$ = $\frac{9}{15}$ = $\frac{54÷9}{90÷9}$ = $\frac{6}{10}$ = $\frac{54÷18}{90÷18}$ = $\frac{3}{5}$ |
| 최대공약수인 18로 나눈 $\frac{3}{5}$ 이 기약분수가 됨|

### 03. 통분
두 수의 분모를 같게 만드는 것
※ 공통분모 : 통분한 분모
> 공통분모가 될 수 있는 수는 두 분모의 공배수

| 예시 |
|---|
| $\frac{2}{6}$ 와 $\frac{3}{8}$ 통분 |
| $\frac{2}{6}$ = $\frac{4}{12}$ = $\frac{6}{18}$ = $\frac{8}{24}$ = $\frac{10}{30}$ = $\frac{12}{36}$ = $\frac{14}{42}$ = $\frac{16}{48}$ ··· |
| $\frac{3}{8}$ = $\frac{6}{16}$ = $\frac{9}{24}$ = $\frac{12}{32}$ = $\frac{15}{40}$ = $\frac{18}{48}$ ··· |
| $(\frac{2}{6} , \frac{3}{8})$ -> $(\frac{8}{24} , \frac{9}{24})$, $(\frac{16}{48} , \frac{18}{48})$ ··· |
