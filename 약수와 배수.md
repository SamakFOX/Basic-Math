# 약수와 배수
---

### 01. 약수와 배수의 정의
약수 : 어떤 수를 나누어 떨어지게 하는 수  
배수 : 어떤 수를 n배 한 수  
> A x B = C 일때 A,B는 C의 약수이며 C는 각각 A와 B의 배수이다.

| | 예시 |
|---|---|
| 8의 약수 | 8 ÷ 1 = 8, 8 ÷ 2 = 4, 8 ÷ 4 = 2, 8 ÷ 8 = 1 |
| | 1, 2, 4, 8 |
| 2의 배수 | 2 x 1 = 2, 2 x 2 = 4, 2 x 3 = 6, 2 x 4 = 8 |
| | 2, 4, 6, 8 ··· |

### 02. 공약수와 최대공약수
공약수 : 두개 이상 자연수의 공통된 약수
최대공약수 : 공약수 중 가장 큰 약수
> 최대공약수의 약수는 공약수와 같음
서로소 : 최대공약수가 1인 두 자연수

| 예시 | 12, 18 |
|---|---|
| 12의 약수 | 1, 2, 3, 4, 6, 12 |
| 18의 약수 | 1, 2, 3, 6, 9, 18 |
| 공약수 | 1, 2, 3, 6 |
| 최대공약수 | 6 |

※ 소인수분해로 최대공약수 구하는 방법  
1) 두 수를 소인수분해  
> 12 = 2² × 3¹  
> 18 = 2¹ × 3²  
2) 공통으로 포함된 소인수의 최소 지수 선택
> 12는 2², 18은 2¹ → 공통은 2¹  
> 12는 3¹, 18은 3² → 공통은 3¹  
> 2¹×3¹ = 2×3 = 6

| 문제풀이 핵심단어 |
|---|

가능한 많이, 똑같이, 나누어줌 등의 표현  
ex) 사탕 12개와 초콜릿 18개를 최대한 많은 사람에게 똑같이 나누어 주려고 할때 인당 몇개씩 나눠줘야 하는가?  
-> 12와 18의 최대공약수인 6명에게 나누어 줄 수 있음.

### 03. 공배수와 최소공배수
공배수 : 두개 이상 자연수의 공통된 배수  
최소공배수 : 공배수 중 가장 작은 배수  
> 공배수는 최소공배수의 배수  

| 예시 | 12, 18 |
|---|---|
| 12의 배수 | 12, 24, 36, 48, 60, 72 ··· |
| 18의 배수 | 18, 36, 54, 72, 90, 108 ··· |
| 공배수 | 36, 72, 108, 144 ··· |
| 최소공배수 | 36 |

※ 소인수분해로 최소공배수 구하는 방법  
1) 두 수를 소인수분해  
> 12 = 2² × 3¹  
> 18 = 2¹ × 3²  
2) 공통, 비공통 포함해서 최고 지수 선택  
> 2의 최고지수는 2², 3의 최고지수는 3²  
> 2²×3² = 4×9 = 36  
>
> 12와 30을 소인수분해시  
> 12=2x2x3  
> 30=2x3x5  
> 공통 비공통 포함 2²x3¹x5¹ = 60으로 최소공배수는 60
